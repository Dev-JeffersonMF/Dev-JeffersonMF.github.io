01 — Sobre Mim

Código com propósito e precisão

Sou um desenvolvedor Full-Stack de Matozinhos, MG, com foco em criar sistemas que resolvem problemas reais de negócios. Não apenas escrevo código — entrego soluções completas, do backend robusto à interface que encanta o usuário.

Tenho experiência no desenvolvimento de SaaS completos para gestão de pizzarias, aplicativos de análise financeira para motoristas, sistemas de fidelização gamificados, e muito mais. Cada projeto é tratado como se fosse o meu próprio negócio.

Meu diferencial está na visão de negócio aliada à técnica: entendo o problema, projeto a solução e entrego com qualidade.
